# Currency Converter

A simple **Currency Converter** built with **React**, **Redux Toolkit**, **Tailwind CSS**, and **shadcn/ui** components.

---

## Features

- Convert between multiple currencies.
- Fetch latest currency rates from API.
- Modern UI using Tailwind + shadcn/ui components.
- State management with Redux Toolkit.
- Responsive design for mobile and desktop.

---

## Tech Stack

- **React** (v18+)
- **Redux Toolkit** (state & async operations)
- **Tailwind CSS** (v4)
- **shadcn/ui** (UI components)
- **Vite** (fast development server & build)

---

## Installation

1. Clone the repo:
```bash
git clone https://github.com/walaa2020/currencies-application.git
```
2. Navigate to project folder
```bash
cd currencies-application
```
3. Install dependencies
```bash
npm install
```
4.Start development server
```bash
npm run dev
```
